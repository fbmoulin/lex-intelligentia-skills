# Academic Writer BR — Produção Científica Jurídica Brasileira

## Normas ABNT Essenciais para o Direito

### NBR 10520 — Citações

**Citação direta curta (até 3 linhas):**
```
Conforme [SOBRENOME] ([ano], p. XX), "[trecho literal entre aspas]".
```

**Citação direta longa (mais de 3 linhas):**
```
Nesse sentido, [SOBRENOME] ([ano], p. XX) preleciona:

    [Recuo de 4 cm da margem esquerda, sem aspas, espaço simples,
    fonte 10, o parágrafo inteiro do texto citado]
```

**Citação indireta (paráfrase):**
```
Segundo [SOBRENOME] ([ano]), [ideia parafraseada].
[SOBRENOME] ([ano]) sustenta que [ideia parafraseada].
```

**Citação de jurisprudência:**
```
[TRIBUNAL], [Órgão Julgador], [Tipo] n. [número], rel. [Min./Des. Nome], j. [data].

Exemplo:
BRASIL. Superior Tribunal de Justiça. Recurso Especial n. 1.251.331/RS, 
rel. Min. Maria Isabel Gallotti, Segunda Seção, julgado em 28.8.2013.
```

### NBR 6023 — Referências Bibliográficas

**Livro:**
```
SOBRENOME, Nome. Título: subtítulo. Edição. Local: Editora, Ano.
```

**Artigo em periódico:**
```
SOBRENOME, Nome. Título do artigo. Nome da Revista, Local, v. [vol.], 
n. [num.], p. [início-fim], mês ano.
```

**Capítulo de livro:**
```
SOBRENOME, Nome. Título do capítulo. In: SOBRENOME, Nome (org./coord.). 
Título do Livro. Ed. Local: Editora, Ano. p. [início-fim]. 
```

**Dissertação / Tese:**
```
SOBRENOME, Nome. Título. Ano. [nº f.] Dissertação (Mestrado em [área]) / 
Tese (Doutorado em [área]) — [Instituição], [local], ano.
```

**Legislação:**
```
BRASIL. [Lei / Decreto / CF] nº XXX, de DD de mês de AAAA. Ementa. 
Publicado em: Diário Oficial da União, Brasília, DD mês AAAA. [Seção X, p. Y].
```

**Acórdão:**
```
BRASIL. [Tribunal]. [Tipo recursal] n. [número], [Órgão Julgador]. 
Relator: [Min./Des. Nome Completo]. [Cidade], [data]. Disponível em: 
[URL]. Acesso em: [data de acesso].
```

## Estrutura de Artigo Científico Jurídico

### IMRAD Adaptado para o Direito

```
TÍTULO
[Informativo, com palavras-chave, até 15 palavras]

RESUMO (150-250 palavras)
Objetivo | Metodologia | Resultados principais | Conclusão

PALAVRAS-CHAVE (3-6 termos)
[Usar termos do vocabulário controlado: ABEC-CAPES, Tesauro Jurídico]

ABSTRACT (tradução do resumo)
KEYWORDS (tradução das palavras-chave)

1. INTRODUÇÃO
   - Contextualização do problema
   - Justificativa da pesquisa (relevância jurídica e social)
   - Hipótese ou problema de pesquisa
   - Delimitação do tema
   - Objetivos (geral e específicos)
   - Estrutura do artigo

2. DESENVOLVIMENTO (subdivisões específicas do tema)
   2.1 [Marco teórico / doutrina]
   2.2 [Marco normativo / legislação]
   2.3 [Marco jurisprudencial]
   2.4 [Análise crítica / proposta]

3. CONSIDERAÇÕES FINAIS
   - Retomada da hipótese
   - Síntese dos argumentos
   - Resposta ao problema de pesquisa
   - Contribuição ao campo
   - Limitações e sugestões de pesquisa futura

REFERÊNCIAS
[Ordenadas alfabeticamente pelo sobrenome do primeiro autor]
```

## Metodologia de Pesquisa Jurídica

### Tipos de Pesquisa em Direito

| Tipo | Quando usar | Técnica |
|------|-------------|---------|
| Dogmática | Interpretação de normas | Hermenêutica jurídica |
| Zetética | Questionar fundamentos | Análise crítica interdisciplinar |
| Empírica | Dados sobre o sistema | Análise quantitativa/qualitativa |
| Comparativa | Diferentes ordenamentos | Direito comparado |
| Histórica | Evolução jurídica | Análise histórico-sociológica |

### Critérios de Cientificidade Jurídica

O artigo jurídico científico deve demonstrar:

```
1. PROBLEMA DEFINIDO
   → Pergunta específica, não opinião genérica

2. HIPÓTESE TESTÁVEL  
   → Afirmação que pode ser confirmada ou refutada

3. METODOLOGIA EXPLÍCITA
   → "A pesquisa é dogmático-jurídica, com análise 
      bibliográfica e jurisprudencial..."

4. REFERÊNCIAS VERIFICÁVEIS
   → Toda afirmação tem fonte; fontes existem e são acessíveis

5. CONTRIBUIÇÃO ORIGINAL
   → O que este artigo acrescenta ao debate?
```

## Vocabulário Científico para o Direito

### Verbos de Citação por Posição Autoral

**Concordância com o citado:**
- "Corrobora tal entendimento [AUTOR] ao afirmar que..."
- "[AUTOR] confirma que..."
- "Em consonância com essa perspectiva, [AUTOR] aponta..."

**Apresentação neutra:**
- "[AUTOR] sustenta que..."
- "Na visão de [AUTOR]..."
- "Para [AUTOR]..."
- "[AUTOR] preleciona..."

**Crítica ao citado:**
- "Divergindo de [AUTOR], entende-se que..."
- "Embora [AUTOR] afirme X, consideramos que..."
- "Tal posicionamento é questionável quando..."

|---------|--------|------|--------------------------|

| Revista de Processo | A1 | Processual | 4-6 meses |

### Checklist de Submissão

- [ ] Normas do periódico verificadas (formato, extensão, espaçamento)
- [ ] Anonimato garantido (nenhuma auto-referência no texto principal)
- [ ] Resumo e abstract dentro do limite de palavras
- [ ] Palavras-chave em PT + EN
- [ ] Referências no padrão ABNT exigido pela revista
- [ ] Declaração de originalidade / não-submissão paralela
- [ ] Dados dos autores apenas na página de rosto

## Formatação ABNT NBR 14724

### Parâmetros de Formatação

```
Papel: A4 (210 x 297 mm)
Margens: Superior 3cm | Inferior 2cm | Esquerda 3cm | Direita 2cm
Fonte: Times New Roman 12 | Arial 12
Espaçamento: 1,5 (texto) | Simples (citações longas, notas, referências)
Parágrafo: 1,25 cm de recuo na primeira linha
Paginação: Superior direita, a partir da página 1
Títulos: Maiúsculas + Negrito (seções primárias)
         Negrito (seções secundárias)
         Itálico (seções terciárias)
```

## Resumos e Abstracts Otimizados

### Fórmula para Resumo Jurídico Científico

```
[Contexto/Problema]: O presente artigo analisa [tema] ante [contexto].
[Objetivo]: Objetiva-se [verbo no infinitivo: examinar/analisar/demonstrar/propor].
[Metodologia]: Para tanto, emprega-se a metodologia [tipo], com análise [bibliográfica/documental/jurisprudencial].
[Resultado]: A pesquisa revela/demonstra/conclui que [resultado principal].
[Conclusão/Contribuição]: Conclui-se, portanto, que [conclusão + contribuição ao campo].
```

**Exemplo:**
```
O presente artigo analisa a aplicabilidade da inteligência artificial na 
fundamentação de decisões judiciais à luz do art. 489, §1º, do CPC/2015. 
Objetiva-se examinar se decisões geradas ou auxiliadas por IA atendem 
ao dever constitucional de fundamentação. Emprega-se metodologia 
dogmático-jurídica, com análise bibliográfica e jurisprudencial dos 
Tribunais Superiores. A pesquisa demonstra que a IA pode auxiliar na 
estruturação decisória sem substituir o juízo valorativo do magistrado. 
Conclui-se que a validação humana é requisito de validade constitucional 
das decisões judiciais algoritmicamente assistidas.
```

## Workflow de Produção Acadêmica

```
1. PROBLEMA → Formular a pergunta científica
2. HIPÓTESE → Qual é minha resposta provisória?
3. PESQUISA → Bibliográfica + jurisprudencial + legislativa
4. PKM → Criar notas atômicas (usar lex-pkm-juridico)
5. OUTLINE → Estrutura IMRAD + desenvolvimento
6. RASCUNHO → Escrever sem autocensura
7. REVISÃO → Argumento + citações + ABNT
8. ABSTRACT → Último a escrever (síntese do produto final)
9. SUBMISSÃO → Formatar para o periódico alvo
```
